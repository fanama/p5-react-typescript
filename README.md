# Express-React-typescript

## install

> npm install

## Build

1. Build the Backend
    - >npm run build-server

2. build the Frontend
    - >npm run build-front
  
the build Project will be in *./server*

## Develop

1. Build the Backend
    - >npm run dev

2. build the Frontend
    - >npm run front-dev

## test the server after build

- >node server/index.js
